### Prerequisites
 
Dnsmasq, Hostapd, Dhcp-Server


```
apt update
apt install dnsmasq
apt install hostapd
apt install isc-dhcp-server
```
### Integrated Tools

```
Sslstrip
Dnsmasq
Driftnet
Tshark
Tcpdump
```

### Starting 

```
git clone github.com/miguelebf/UnicornAp
cd UnicornAp
python3 unicornAp.py
```
