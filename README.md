# UnicornAp

A little script in python to create a fake AP


### Prerequisites
 
Dnsmasq

```
apt update
apt install dnsmasq
```
### Integrated Tools

```
Sslstrip
Dnsmasq
Driftnet
Tshark
Tcpdump
```

### Starting 

```
git clone github.com/miguelebf/UnicornAp
cd UnicornAp
python3 unicornAp.py
```

###Tested on 

-Kali Linux 2017.3
